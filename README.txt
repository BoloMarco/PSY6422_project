# PSY6422_project

# Overview

This project makes an animated GIF with two line graphs: one graph represents the “New HIV diagnoses in the UK by probable route of exposure”, while the other graph represents “New HIV diagnoses in the UK by heterosexual contact”.
Data from the UK Health Security Agency that can be found on the site gov.uk. Details here:
https://assets.publishing.service.gov.uk/government/uploads/system/uploads/attachment_data/file/1037993/hiv-country-region-data-tables-2021.ods
